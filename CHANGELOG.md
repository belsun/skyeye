# Changelog

All notable SkyEye product changes are tracked here so the project reads like an
active, iterative product rather than a one-off dashboard.

## 2026-06-04

### Added

- Added this changelog and linked it from the README so future bug fixes and
  product iterations have a visible release trail.
- Added a stronger English GitHub product narrative focused on SkyEye as a
  news-to-price market intelligence tracker.
- Added bilingual positioning to the public project description, including
  current Chinese/English support and the plan for future language packs.

### Changed

- Updated the real product showcase image script so GitHub introduction images
  use English headlines and demo copy.
- Refined non-equity instrument pages so indices, crypto pairs, and commodities
  display quote/tracker context instead of company-profile language.

### Fixed

- Fixed misleading company-profile wording for instruments such as `GC=F`,
  `BTC-USD`, and `^HSI`.

## 2026-06-03

### Added

- Added real UI-based product showcase graphics for GitHub and application
  materials.
- Added 15-second launch-video assets and cover images.
- Added multi-theme presentation support, including brighter product styles.

### Changed

- Reworked the deep symbol view so company news sits below the K-line while
  policy, industry, macro, and external-impact items sit in the right panel.
- Refined showcase titles to make the K-line workflow the first product story.

### Fixed

- Locked the local development server to loopback by default to reduce accidental
  exposure on the local network.
