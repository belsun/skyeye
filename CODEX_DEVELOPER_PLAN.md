# Codex Developer Program Application Notes

GitHub user: `belsun`

Project: `SkyEye`

SkyEye is a full-stack financial research dashboard focused on turning market news, IPOs, industry-chain events, and technical signals into a paper-trading research loop.

## What Codex Helped Build

- Refactored the product direction from a generic dashboard into an Opportunity Radar.
- Added dual-currency paper trading for HKD and USD books.
- Improved Hong Kong ticker handling, including five-digit HKEX code normalization.
- Built beginner-facing explanations for risk, news impact, and paper-only trade setup.
- Added API tests for radar scoring, book routing, search aliases, and cash rejection.

## Why It Fits

This is an open-source, learning-oriented project that demonstrates Codex working across:

- Python/FastAPI backend APIs
- SQLite schema and data workflows
- React/TypeScript frontend
- Financial research logic
- Test coverage and product documentation

## Public-Safety Positioning

SkyEye does not place real trades, does not connect to brokers, and labels trade actions as simulated paper trading. It is intended for research, education, and product experimentation.
